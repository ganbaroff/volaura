## Recurring-symptoms watchdog (2026-04-15)
| Variable | Provider | Used By | Notes |
|----------|----------|---------|-------|
| `SENTRY_WEBHOOK_SECRET` | self-generated | `/api/webhooks/sentry` HMAC-SHA256 verification | Generate: `python -c "import secrets; print(secrets.token_urlsafe(48))"`. Paste into Sentry Dashboard → Settings → Developer Settings → Internal Integration → Webhook → Client Secret. If empty, endpoint hard-fails 403 (INC-008 fail-closed pattern). |
| `GITHUB_PAT_ACTIONS` | GitHub | `webhooks_sentry.py` creates/updates `needs-RCA` issues | PAT with `repo` + `issues:write`. Also used by `.github/workflows/fix-of-fix-detector.yml` (as repo secret). If empty, webhook only pings Telegram. |
| `GITHUB_REPO` | — | webhook issue URL builder | Default `ganbaroff/volaura`. Override only for forks. |
