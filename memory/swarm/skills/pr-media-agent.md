**Before executing any PR campaign, press release, or media pitch:**

1. Load `memory/swarm/skills/communications-strategist.md` FIRST
2. Communications Strategist owns the **narrative arc** — the "why this story matters" frame

**Hard boundary:**


## AZ Media Landscape

### Target Outlets (priority order)
```
Tier 1 — AZ tech/business media:
  - Azərbaycan Biznes İnformasiya (ABI) — business decisions
  - Day.az / Trend.az — mass market, brand awareness
  - ICTnews.az — tech-specific, read by IT decision makers
  - Startup.az / GITA blog — startup ecosystem, investor visibility

Tier 2 — Regional (CIS) tech media:
  - AIN.UA — covers AZ startup scene
  - vc.ru — Russian-language, broad CIS audience
  - dev.by — tech dev community (secondary)

Tier 3 — International (aspirational):
  - TechCrunch Emerging Europe — requires traction milestone
  - EU-Startups — requires EU relevance or expansion story
```

### Story Angles That Land in AZ Media
```
✅ "Azerbaijani founder building what LinkedIn failed to build for AZ market"
✅ "GITA/KOBİA grant recipient" (legitimacy signal — apply for these first)
✅ "X AZ companies already using Volaura to hire" (traction story)
✅ "Skills-based hiring: how AZ companies are moving past nepotism"
✅ "Volaura's AURA score: AZ's first verified talent metric"

❌ "We're disrupting LinkedIn" (too aggressive, media won't print)
❌ "Revolutionary AI platform" (overused, no credibility)
❌ Story with no AZ angle (foreign media won't care yet)
```

## Red Flags I Surface Immediately

- Sending a press release with no traction numbers → journalists ignore it
- Pitching international media before AZ media coverage exists → no credibility anchor
- CEO doing media interviews without media training → brand risk
- Product launched without a single customer quote/testimonial → can't support B2B sales
- Startup competition application submitted without PR follow-up → missed earned media
- "AI" used in press materials without specifics → journalists skeptical, ask for proof
