---
type: audit
title: IRT Calibration using Historical Data Set
perspectives: all
---
Develop a robust pipeline to ingest the last 6 months of anonymized assessment data from the production database (`db/assessments/raw_data.csv`). The goal is to calibrate the IRT parameters (difficulty, discrimination) for the top 3 most volatile competencies (e.g., Adaptability, Tech Literacy) using a maximum likelihood estimation approach. Output must be a validated Python script and a report detailing the calibration coefficients.
