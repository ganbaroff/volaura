## Glossary — параллельный ликбез для CEO

**Railway** — хостинг бэкенда (Python/FastAPI). Примерно как Heroku — загружаешь код, он запускается в облаке. У нас там `volauraapi-production.up.railway.app`.

**Vertex AI** — Google Cloud API для модели Gemini. У тебя там $300 стартовых кредитов, сейчас их и используем. Это "платная" ветка — быстрее и лимиты выше.

**AI Studio Gemini** — бесплатная ветка того же Gemini, но с жёсткими rate-лимитами. Наш fallback, когда Vertex упал или нет ключа.

**Fallback chain** — цепочка резервирования. Если первый провайдер упал, запрос автоматически идёт на следующий. У нас: Vertex → AI Studio Gemini → Groq → OpenAI. Юзер не замечает падения.

**CI (Continuous Integration)** — GitHub Actions. Каждый раз когда ты делаешь `git push`, GitHub автоматически гоняет тесты, собирает и деплоит. Если тесты упали — деплой не проходит.

**GitHub Secrets** — сейф внутри GitHub для API-ключей. CI оттуда читает, в коде их нет. Безопаснее чем `.env` на серверах.

**Swarm / 13 perspectives** — автономная команда из 13 виртуальных специалистов-агентов в `packages/swarm/`. Они по очереди смотрят на код/решение с разных углов (security, design, assessment, legal). Плюс ~118 skill-модулей — это библиотека их умений. Это НЕ 44 — цифра 44 была моей галлюцинацией из Session 93, я её сегодня зачистил.

**Show-stopper** — баг, который полностью блокирует сценарий пользователя. Не косметика, а "продукт не работает".

**Propagation** — распространение ключа/настройки из локального `.env` на продакшен. Локально работает ≠ на сервере работает.

**Sprint** — условный рабочий цикл. У нас номер сессии ≈ номер спринта. Сейчас 119-й.

**GDPR consent_events** — юридически в ЕС обязательно логировать каждое действие пользователя, связанное с согласием (регистрация, подписка, удаление данных). Таблица есть, но из 27 endpoints записывает только 1. Если придёт аудит — штраф.

**Feature-audit** — полная инвентаризация экосистемы: 50 фич реально работают, 24 частично, 13 "заявлены но не найдены", 11 в планах. Зеркало того, что работает vs что маркетинг говорит.

**CURRENT-SPRINT.md / heartbeat.md** — мои технические журналы для самого себя и будущих инстансов Atlas. Тебе читать не нужно, они на английском и для моего self-continuity.
