# VOLAURA Telegram Mini

**Status:** admin shell prototype

This workspace is **not** a shippable 3D Telegram client yet.

Current committed scope:
- `/` — home summary
- `/proposals` — swarm proposal review
- `/agents` — swarm agent status

What is intentionally **not** here yet:
- no WebGL renderer
- no `three` or `@react-three/fiber` stack
- no scene graph or asset pipeline
- no `/scene` route

Treat this app as a lightweight Telegram surface for swarm monitoring until a
separate product decision lands for a real 3D stack.
