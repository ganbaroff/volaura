- 0% AC написаны до кода — никогда

**Что сделали:** TASK-PROTOCOL v8.0, QUALITY-SYSTEM.md, 4 новых агента (AC writer, QA verifier, DORA tracker, Langfuse), pre-commit hook, дефектная аутопсия (61 commit → 3 реальные причины).

### 2. Важный момент: CLASS 11

CEO спросил "Langfuse — лучшее решение?" CTO инстинктивно хотел подтвердить то что сам же предложил. CEO остановил. Запустили реальное исследование: 8 инструментов сравнили. Langfuse подтверждён — но теперь с данными, не с мнением.

**Урок:** CTO не может подтверждать собственные предложения. CLASS 11 — self-confirmation bias.

### Что такое VOLAURA

B2B платформа верификации компетенций для рынков AZ/CIS.

**Как работает:**
1. Volunteer/Professional проходит IRT/CAT адаптивную оценку (70 вопросов, 7 компетенций)
2. Получает AURA score (0-100) + badge (Bronze/Silver/Gold/Platinum)
3. Профиль становится виден организациям
4. Организации ищут верифицированный талант по компетенциям и AURA score

**Ключевые метрики которые должны знать все агенты:**

- Stack: FastAPI + Next.js 14 + Supabase
- Hosting: Railway (API) + Vercel (Frontend)
