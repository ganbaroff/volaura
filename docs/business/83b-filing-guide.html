Единственный путь: прямая отправка DHL Express Worldwide из Баку в IRS (230 AZN negotiated). Запасной путь через друга в США ОТПАЛ — друг недоступен на 2026-04-18. Запасного варианта нет — шаг A1 (проверка IRS PDS list) стал hard gate, не рекомендацией.

Почему DHL-прямая работает: IRS по коду §7502(f) и Notice 2016-30 признаёт postmark от «designated private delivery services» наравне с USPS. DHL Express Worldwide и ряд других DHL Express сервисов входят в этот список. Значит DHL air waybill с датой отправки = legal filing date, идентично USPS Certified postmark.

Критичный чек-поинт перед походом: зайди на irs.gov → поиск «Private Delivery Services» → найди актуальный PDS list. Убедись что DHL Express Worldwide (или тот сервис что DHL Баку предлагает) ТАМ. Если его нет в текущем списке — разворачивайся, переходи на friend-path ниже. 5 минут на проверку сэкономят tax disaster на $10K+.

Нотариально заверять не нужно. 83(b) требует только твоей подписи и даты ручкой на форме. Без нотариуса, без apostille, без свидетеля. Stripe Atlas генерирует предзаполненную форму в дашборде.

EIN ещё не получен (expected Apr 29 – May 13 per Stripe Atlas). В поле EIN на форме будет «Applied for» — так и должно быть, IRS принимает. 83(b) по закону не ждёт EIN, правило 30 дней жёсткое.

📮USPS Certified Mail + Return Receipt

На стойке друг говорит (это есть в cover letter):

"I need to send this letter by Certified Mail with Return Receipt."

Две формы:

  • PS Form 3800 — Certified Mail (белая наклейка с бар-кодом + белая квитанция для друга)
  • PS Form 3811 — Return Receipt (зелёная карточка, приклеивается к конверту, вернётся другу после того как IRS распишется)

Адреса:

  • To: IRS адрес из Stripe Atlas пакета (берётся с bundled sheet)
  • From: друга домашний адрес (чтобы stamped-copy и Return Receipt пришли к нему)

Стоимость: ~$9–12 суммарно (Certified $4.40 + Return Receipt $3.85 + First-Class postage $0.68).

Критично: именно Certified Mail with Return Receipt. Не Priority Mail. Не Express. Certified Mail даёт legal proof of filing date — это единственный принятый IRS формат для 83(b).
14

📅Штемпель (postmark) — самое важное

USPS ставит postmark в день подачи. Эта дата = дата filing 83(b) в глазах IRS.

Цель: postmark ≤ 28 апреля 2026. Раньше — лучше.

Если друг подаёт 22–24 апреля — есть 4–6 дней запаса. Safe.

Если вдруг что-то идёт не по плану и ты видишь что время поджимает — в крайнем случае друг может пойти в тот же день как получил пакет. USPS работает Mon–Sat.
