# Beta Invite Message Templates
*Cultural Intelligence Agent recommendation — CEO personalizes before sending.*
*Created: 2026-03-29. Use for Week 1 warm outreach (30 individual contacts).*

---

## Rules (Cultural Intelligence Agent)

1. **Personal reference first** — start with something shared, not with the product
2. **Frame as a favor to Yusif** — not a product pitch. "I'd love your honest feedback" ≠ "check out my startup"
3. **Language by contact background:** AZ native → AZ message; CIS background → RU message; International/EN contact → EN message
4. **Never say "platform"** — say "system" or name the specific thing (assessment, AURA score)
5. **No "excited to share"** — share the real situation ("we have zero users, you'd be the first")
6. **COP29/WUF13 contacts:** reference the shared event specifically. "After COP29" > "I know you from the events sector"

## Code Assignment Guide

| Contacts | Code range | Account type |
|----------|-----------|-------------|
| Individual professionals (volunteers) | BETA_01 – BETA_25 | volunteer (default) |
| LinkedIn CTA (post) | OPEN | volunteer |
| Org design partners (Week 2 Day 4+) | ORG_01 – ORG_02 | organization |
| Reserved (personal friends/family) | BETA_26 – BETA_30 | volunteer |

**Rule:** Assign one code per person. Track in a spreadsheet: contact → code → date sent → registered (Y/N).
Do not reuse codes — each code is a trackable attribution point.

## Sequencing (Sales Strategist recommendation)

**Week 1 Day 1-3:** Send to 15 highest-trust individual contacts. These are people who will actually do the assessment and generate AURA scores.

**Week 1 Day 4-7:** Send to next 15 individual contacts based on Week 1 completion rate.

**Week 2 Day 4+:** Only AFTER 15 individual users have AURA scores with `visible_to_orgs=true`:
- Invite 2 org design partners (qualified: must have active volunteer programs)
- Frame explicitly: "beta design partner, not customer — I need 30 minutes of your feedback"
- Use ORG_01, ORG_02 codes

**Never send org invites before individual users have AURA scores.** An org arriving to 0 searchable profiles will not return.
