## Топ-5 рисков сценария

1. **Scope creep.** Во время Ф1 Yusif добавит новую фичу → Ф2 сорвётся. Mitigation: формальный feature freeze после старта фазы.
2. **Один исполнитель.** 90 дней соло-темп = burnout. Mitigation: swarm берёт 40% работы (bridges, question bank, E2E, docs consolidation — всё можно делегировать агентам).
3. **Bridge contract drift.** Каждый продукт на своём стеке. Mitigation: `packages/ecosystem-contracts/` с Pydantic + zod генерируемыми схемами.
4. **LLM-cost spike.** Ф3+ откроет трафик. Mitigation: daily budget cap + Cerebras primary + Ollama fallback для любых non-critical вызовов.
5. **Constitution поломается без CI.** После Ф2 кода в 5 раз больше. Mitigation: CI-gate в Ф4 — но добавить `constitution_checker` в pre-commit уже сейчас (Ф1).
