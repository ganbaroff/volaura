# Promotion Agency Skill

## When to Use
- Founder asks "what should I post?" or "how do I promote?"
- Building content calendar
- Writing LinkedIn posts, press releases, media pitches
- Personal branding strategy for CEO
- Viral campaign design
- Crisis management

## Pre-Flight Checks (BEFORE creating any content)

```
□ Read TONE-OF-VOICE.md (if exists) — brand voice rules
□ Read mistakes.md — past content failures (e.g., mentioning employers → HR problems)
□ Read patterns.md — "silent contracts" about content
□ Verify: CEO approves all public content before publishing? (most do)
□ Verify: Current product positioning (NOT stale branding)
□ Check: Any sensitive topics to avoid? (investors, competitors, personal life)
```

**CRITICAL GUARDRAIL:** Never fabricate metrics, quotes, or achievements in content. If the startup has 50 users, don't write "thousands." Authenticity > impressiveness.

## LinkedIn Strategy Framework

### Optimal Cadence (data-backed, 2025-2026)

| Frequency | Best For | Expected Engagement |
|-----------|----------|-------------------|
| 2/week | Solo founder, limited time | 3-4% baseline |
| **3/week** | **Standard recommendation** | **4-5% with quality content** |
| 4-5/week | Growth phase, full-time content | 5-6% but requires volume |

**Schedule:** Tue/Wed/Thu, morning (8-9 AM in audience timezone)
**Why not Mon/Fri:** Monday = inbox clearing. Friday = checked out. Weekend = low B2B reach.

**MENA/CIS adjustment:** Sunday-Thursday workweek in some markets. Test Sunday posts if audience is AZ/UAE.

### Content Mix (adapt ratios to stage)

| Type | Ratio | Purpose | Format |
|------|-------|---------|--------|
| Founder storytelling | 35-40% | Build trust, show journey | Carousel, text |
| Educational / value | 30-35% | Demonstrate expertise | Carousel, document |
| Product updates | 15-20% | Drive signups | Image + text |
| Social proof | 5-10% | Credibility | Screenshot, quote |

### Format Performance (2026 benchmarks)

| Format | Avg Engagement | When to Use |
|--------|---------------|-------------|
| **Carousel** | **21.7%** | Educational content, storytelling |
| Document (PDF) | 6.1% | Frameworks, reports, case studies |
| Multi-image | 6.6% | Before/after, product screens |
| Video (<60s) | 5.6% | Founder message, demo |
| Text-only | 3-4% | Hot takes, quick updates |

### Hook Templates (fill in blanks)

```
PATTERN 1 (Contrarian): "[Common belief] is wrong. Here's what actually works."
PATTERN 2 (Vulnerability): "I [failed/struggled] at [X]. Here's what I learned."
PATTERN 3 (Data): "[Surprising number]. That's [what it means for audience]."
PATTERN 4 (Question): "Why do [audience] still [outdated practice]?"
PATTERN 5 (Story): "[Specific moment] changed how I think about [topic]."
```

**Anti-patterns:**
- "I'm excited to announce..." (nobody cares about your excitement)
- Starting with company name (lead with problem/insight)
- Hashtag spam (max 3-5 relevant hashtags)
- Emoji overload (1-2 purposeful, not decorative)

## Personal Branding Framework

### LinkedIn Profile Optimization

```
HEADLINE FORMULA: [Role] | [What you're building] | [Unique angle]
  Good: "CEO @ [Startup] | Making [X] accessible to [audience]"
  Bad:  "Visionary entrepreneur | Thought leader | World changer"

ABOUT SECTION (300 words max):
  Paragraph 1: The problem that keeps you up at night
  Paragraph 2: What you're building to solve it
  Paragraph 3: Your unfair advantage (background, insight, tech)
  Paragraph 4: CTA — what you want readers to do

FEATURED SECTION: Pin 3 items:
  1. Best-performing post (social proof)
  2. Product demo / website link (conversion)
  3. Media mention or talk (credibility)
```

### Thought Leadership Topics (generate for any startup)

```
TOPIC GENERATOR:
1. "Why [industry status quo] is broken" → your product fixes it
2. "The [number] things [audience] gets wrong about [your domain]"
3. "How [emerging trend] is changing [your market]"
4. "Lessons from building [your product] in [emerging market]"
5. "[Your background] taught me [unexpected insight about your industry]"
```

## Viral Mechanics

### K-Factor (correct formula)

```
K-factor = i × c

i = number of invites/shares per user
c = conversion rate of invites to signups

K > 1 = viral growth (each user brings >1 new user)
K = 0.5-1.0 = healthy organic growth
K < 0.5 = paid acquisition needed

EXAMPLE:
  Average user shares badge on LinkedIn: i = 3 (reaches 3 people who click)
  Of those, 10% sign up: c = 0.10
  K = 3 × 0.10 = 0.30 (not viral, but healthy organic)

  With AI video sharing: i = 8 (video reaches more)
  Of those, 15% sign up: c = 0.15
  K = 8 × 0.15 = 1.20 (VIRAL)
```

**⚠️ Common mistake:** K-factor ≠ engagement rate. K-factor measures GROWTH, not likes/comments.

### Viral Loop Design

```
1. USER COMPLETES ACTION → earns shareable artifact (badge, certificate, video)
2. ARTIFACT HAS SOCIAL PROOF → visible score/rank that triggers comparison
3. SHARING IS FRICTIONLESS → 1-click share to LinkedIn/WhatsApp/Telegram
4. VIEWER SEES VALUE → "I want this too" → signs up
5. NEW USER COMPLETES ACTION → loop restarts
```

## Media Outreach

### Press Release Structure

```
HEADLINE: [Company] launches [product/feature] to [solve problem]
SUBHEAD: [Key metric or differentiator]

PARAGRAPH 1 (WHO/WHAT/WHEN/WHERE):
  [Company], [one-line description], today announced [news].

PARAGRAPH 2 (WHY IT MATTERS):
  [Market context]. [Problem statement]. [How this solves it].

PARAGRAPH 3 (QUOTE from founder):
  "[Authentic quote about vision, not marketing speak]"

PARAGRAPH 4 (DETAILS):
  [Product specifics, pricing, availability]

PARAGRAPH 5 (ABOUT):
  [Company] is [description]. Founded in [year] by [founder].
  Learn more at [website].

CONTACT:
  [Name], [Email], [Phone]
```

### Outreach Target Template

```
| Tier | Outlet Type | Pitch Angle | When |
|------|-------------|-------------|------|

| 2 | Industry vertical | "[Industry] disrupted by [approach]" | Post-traction |
| 3 | Global tech | "From [country] to global: [story]" | Post-funding |
```

## Crisis Management

```
SEVERITY 1 (negative comment):
  → Thank for feedback. Offer DM. NEVER delete.

SEVERITY 2 (factual error in your content):
  → Correct immediately. Own the mistake publicly.

SEVERITY 3 (PR crisis — CEO personal, legal, etc):
  → Do NOT respond immediately (24h rule).
  → Draft response. Have advisor review.
  → CEO authentic voice only. No corporate speak.
  → NEVER "no comment." Address or stay silent.

ABSOLUTE RULE: Never mention employers, partners, or third parties
without their explicit permission. This has caused real harm before.
```
