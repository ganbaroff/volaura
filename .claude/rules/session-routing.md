# Session Routing

## Session Start
Read `.claude/breadcrumb.md` first. It has the current state. Then `memory/context/sprint-state.md` if needed.

## How to Route
- "продолжи" / "continue" → read breadcrumb, continue from where we stopped
- New task → research first, then build
- Question → no code, just answer
- Content request → external model writes, CTO reviews

## Session End
Update breadcrumb.md with current state. Update sprint-state.md if something significant shipped.

## Don't
- Don't write code without understanding context
- Don't work alone on non-trivial tasks
- Don't use Claude as subagent — use external LLMs
