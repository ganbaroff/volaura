# Environment Variables Reference

All keys stored in `apps/api/.env` (git-ignored). For production: use Railway Secrets.

## Required for MVP
| Variable | Provider | Used By | Free Tier Limit |
|----------|----------|---------|-----------------|
| `GEMINI_API_KEY` | Google AI | BARS, Embeddings, Coach | 15 RPM, 1M tok/day |
| `SUPABASE_URL` | Supabase | All DB ops | Free tier |
| `SUPABASE_SERVICE_KEY` | Supabase | Backend admin | — |
| `SUPABASE_ANON_KEY` | Supabase | Frontend auth | — |

## Optional (allocated by swarm DSP)
| Variable | Provider | Used By | Free Tier Limit |
|----------|----------|---------|-----------------|
| `OPENAI_API_KEY` | OpenAI | Anti-gaming only | PAID — set $5/mo cap! |
| `GROQ_API_KEY` | Groq | Swarm, notifications | 14,400 req/day |
| `DEEPSEEK_API_KEY` | DeepSeek | Fallback, ranking | $0.14/MTok |
| `OPENROUTER_API_KEY` | OpenRouter | Free models fallback | Varies |

## Security Checklist
- [ ] OpenAI billing cap set to $5/mo in dashboard
- [ ] GCP quota set for Gemini API
- [ ] Railway Secrets used (not .env file upload)
- [ ] Separate dev/prod keys created
- [ ] Supabase service key rotated after initial dev
