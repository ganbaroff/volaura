# Video Format Reference

## YouTube Long-Form (8-15 minutes)

### Structure

```
HOOK (0:00-0:15)
  → Open with the strongest moment. Not "Hi, I'm Yusif."
  → First 3 seconds decide if viewer stays.
  → Pattern: surprising fact, bold claim, or mid-action moment.

CONTEXT (0:15-1:30)
  → Why this matters. Set the stakes.
  → "If you work with volunteers, this changes everything."
  → Introduce the specific problem.

STORY ARC (1:30-8:00)
  → The meat. Use narrative structure:
    - Situation → Complication → Resolution
    - Each section ~2 min max before a visual/tonal shift
  → Include "open loops" — tease what's coming to retain viewers.
  → Every 2 minutes: pattern interrupt (graphic, B-roll, screen recording, tone shift).

KEY INSIGHT (8:00-10:00)
  → The "aha" moment. What the viewer takes away.
  → This is the screenshot-worthy quote.
  → Make it specific: "47 agents evaluated 90 questions in 6 seconds" not "AI is powerful."

CTA + CLOSE (10:00-12:00)
  → ONE action. Not "like subscribe comment share follow join."
  → "Link in description. Try your AURA score."
  → Or: "Comment with your experience. I read every one."
  → End strong. Last sentence should be memorable.
```

### Script Writing Rules

**Tone:**
- Conversational, not presentational. Write like talking to one smart friend.
- Short sentences. 8-12 words average.
- Paragraphs = 2-3 sentences max (this is a script, not an essay).
- Read it aloud. If it sounds stiff → rewrite.

**Visual Cues (embed in script):**
```
[SCREEN: dashboard showing AURA scores]
[CUT TO: Yusif at desk, frustrated face]
[GRAPHIC: "47 agents. 0 users." text on black]
[B-ROLL: volunteers at WUF13 event]
```

**Bilingual Strategy for YouTube:**
- Primary: RU or EN (choose based on topic and target audience)
- AZ subtitles always included
- If topic is AZ-specific (volunteer culture in Baku): film in AZ, subtitle EN/RU
- YouTube SEO: title + description in primary language, tags in all 3

### YouTube SEO Elements

```
TITLE: [Number or Hook] + [Topic] + [Qualifier]
  Good: "44 AI Agents Run My Startup (Here's What Happened)"
  Good: "I Built a GRE for Volunteers — The Results Were Surprising"
  Bad: "VOLAURA Product Update Q2 2026"

DESCRIPTION (first 2 lines visible):
  Line 1: Expand on the hook (not repeat the title)
  Line 2: CTA with link
  Line 3+: Timestamps, links, context

TAGS: 15-20 tags. Mix broad ("AI startup") + specific ("volunteer assessment tool") + long-tail ("how to evaluate volunteers with AI")

THUMBNAIL BRIEF:
  - 3 elements max: face + text + one object/graphic
  - Text: 4-6 words, readable on mobile
  - High contrast. No red (Constitution: Never Red).
  - Face with clear emotion (shock, curiosity, determination)
  - Test: shrink to phone size. Still readable?
```

## TikTok / YouTube Shorts (15-60 seconds)

### Structure

```
HOOK (0:00-0:03)
  → THE most important 3 seconds.
  → Start mid-action or with a bold statement.
  → "I gave 44 AI agents one task." (pause)
  → "This app knows your volunteer skills better than you do."
  → NEVER start with "Hey guys" or "So today I wanted to..."

TENSION (0:03-0:20)
  → Build curiosity. What happened? What went wrong?
  → Show, don't tell. Screen recordings > talking head.
  → Quick cuts. 2-3 second clips.

PAYOFF (0:20-0:45)
  → Deliver the insight.
  → Make it shareable — viewer should want to send to someone.
  → "And that's how an AI scored higher than human evaluators."

CTA (0:45-0:60)
  → Soft CTA only. No "like and subscribe" begging.
  → "Link in bio" or "Comment if you've seen this."
  → Or: end on the strong moment, no CTA at all (TikTok rewards watch-through).
```

### TikTok Script Rules

**Length by type:**
| Type | Duration | Best for |
|------|----------|----------|
| Hot take | 15-20 sec | Contrarian opinions, quick insights |
| Mini-story | 30-45 sec | "Day in the life", behind-the-scenes |
| Explainer | 45-60 sec | How something works, product demo |
| Reaction | 15-30 sec | Responding to trends, comments |

**Format Options:**
- **Talking head** — Yusif speaks directly to camera. Best for Series 2 (personal voice).
- **Screen recording + voiceover** — Show the product. Best for demos.
- **Text overlay + music** — Cinematic/aesthetic. Best for brand pieces.
- **Green screen** — React to headlines, screenshots, data. Good for tech commentary.

**AZ-specific TikTok:**
- AZ TikTok audience is younger (18-25), mobile-first
- Humor works better than authority in AZ
- Use AZ slang naturally — not formal literary Azerbaijani
- Reference local context: Baku landmarks, CIS culture, volunteer culture
- AZ captions even when speaking AZ (many watch on mute)

**Hashtag Strategy:**
- 3-5 hashtags per video
- Mix: 1 trending + 2 niche + 1 branded (#volaura or #aurascore)
- AZ hashtags: #baku #azerbaijan #könüllü (volunteer) #texnologiya
- EN hashtags: #aiagents #volunteertech #startup #techfounder

## Cross-Platform Repurposing Rules

One video recording can produce:

```
1 YouTube long-form (8-15 min)
  ↓ extract best 3 moments
3 TikTok/Shorts (30-60 sec each)
  ↓ transcribe + rewrite
3 LinkedIn posts (EN, AZ, Series 1)
  ↓ pull key quote
1 Carousel (8 slides from key points)
  ↓ grab thumbnail frame
1 Thumbnail (from best expression frame)
  ↓ clip audio
1 Podcast segment (audio-only, 5-8 min)
```

Total from 1 recording session: ~10 content units.
