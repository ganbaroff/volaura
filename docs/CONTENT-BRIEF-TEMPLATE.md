## CULTURAL GUARDRAILS CHECKLIST

**Before copywriter starts, check all 7 principles from TONE-OF-VOICE.md:**

### P1 — Earned Authority
- [ ] Every claim can be sourced to: a specific event, a date, or a measurable outcome
- [ ] No vague credentials ("extensive experience," "deep understanding")
- [ ] Evidence is concrete: names, dates, numbers, roles

**Action if FAIL:** Rewrite claims to be specific. Remove anything that can't be sourced.

---

### P2 — Asymmetric Transparency
- [ ] Disclose something most competitors hide (but not everything)
- [ ] Disclosure is surprising, not exhaustive
- [ ] Not a list of "lessons I learned" — that reads as performative

**Action if FAIL:** Remove oversharing. Ask: "Why is THIS disclosure valuable?" If no answer, cut it.

---

### P3 — Structural Observation, Not Personal Grievance
- [ ] Post names a broken SYSTEM, not broken PEOPLE
- [ ] No implied targeting of specific past employers or individuals
- [ ] Reads as "this pattern exists" not "these people messed up"

**Action if FAIL:** Remove any reference that could be read as naming someone. Reframe as systemic.

---

### P4 — Show the Machine
- [ ] AI involvement is specific and functional (Claude as CTO, making decisions, being disagreed with)
- [ ] Not vague ("AI-powered," "built with AI," "cutting-edge")
- [ ] Shows HOW Claude is used, not just THAT Claude is used

**Action if FAIL:** Specify the functional role. Remove marketing noise.

---

### P5 — The Question Is the Hook
- [ ] Post ends with a question that requires response cost (reflection, memory reconstruction)
- [ ] NOT "What do you think?" or "Can you relate?" (zero cost)
- [ ] Reader must recall a specific experience to answer honestly

**Action if FAIL:** Rewrite question to demand memory reconstruction, not opinion.

---

### P6 — One Voice Per Series
- [ ] Series 1: Claude's voice only (Yusif in 3rd person, from Claude's perspective)
- [ ] Series 2: Yusif's voice only (no Claude appearing as first-person narrator)
- [ ] No voice bleed between series

**Action if FAIL:** Remove sentences that break character. Rewrite in consistent voice.

---

### P7 — Real Numbers or No Numbers
- [ ] If a number appears, it comes from a real, nameable source
- [ ] No approximations ("dozens," "hundreds," "multiple," "many")
- [ ] Specific numbers signal "I was actually there"

**Action if FAIL:** Replace with exact number from source, or remove number entirely.

## FORBIDDEN PATTERNS

**Do NOT use these phrases. They appear in every startup. Zero signal.**

| Forbidden phrase | Why banned | Use instead |
|------------------|-----------|-------------|
| "AI-powered platform" | Every startup says this | Specific: "Built with Claude as my AI CTO — making real architectural decisions" |
| "ecosystem" | Cliché signaling no real customers yet | Name the parties: volunteers, orgs, coordinators |
| "leverage" | Corporate filler | Specific verb: measure, compare, evaluate, assign |
| "passionate about" | Self-reported, unverifiable | Show the work. Passion is demonstrated, not declared |
| "innovative solution" | Meaningless without specifics | Name what changes: "selection shifts from connection-based to score-based" |
| "stay tuned" | "I have nothing to say right now" | Only publish when you have something real to say |
| "excited to share" | Filler that delays the hook | Start with situation or problem |
| "DM me to learn more" | Begging | Specific CTA: email or specific question to answer |
| "scaling" (abstract) | "I want to grow" — empty | Name markets: Tbilisi, Almaty, Istanbul |
| "thought leader" | Self-assigned title | Thought leaders are named by others |
