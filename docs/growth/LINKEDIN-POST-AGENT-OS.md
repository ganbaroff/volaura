I built a self-improving operating system for Claude.

Not a prompt template. An actual OS that makes AI agents better at decisions — and learns from its own mistakes.

The core: Decision Simulation Protocol (DSP).

Before any significant decision, 6 AI personas debate alternatives. An Attacker finds exploits. A Scaling Engineer asks "what breaks at 10x?" Each scores paths on 5 dimensions. Winner must clear a ≥35/50 confidence gate.

After execution, the system compares predictions vs reality. If a persona's concerns proved right — its influence weight increases. The system evolves its own rules.
