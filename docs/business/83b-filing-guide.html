🖨️Распечатать 2 копии 83(b) election

A4, белая бумага, чёрный картридж, одна сторона листа.

Две копии. Одна уйдёт в IRS. Вторую IRS проштампует и вернёт другу (друг потом тебе перешлёт). Третью копию-скан сделаем на этапе сохранения.

Если дома принтера нет — любая "типография"/"kopi merkezi" на улице. Цена 30-50 qəpik за лист.

🖊️Подписать обе копии СИНЕЙ или чёрной ручкой

Обычная шариковая. Не карандаш. Не красная. Не маркер.

Места для подписи:

  • Signature line — твоя подпись
  • Print name — печатными буквами: YUSIF GANBAROV
  • Date — сегодняшнее число в формате 04/18/2026 или 04/19/2026 (день подписи, не назад)
Дата подписи = день, когда реально подписываешь. Никогда не ставь дату задним числом — это уголовка в IRS (backdating). Любая дата с сегодня по 28 апреля — легальна.

📮USPS Certified Mail + Return Receipt

На стойке друг говорит (это есть в cover letter):

"I need to send this letter by Certified Mail with Return Receipt."

Две формы:

  • PS Form 3800 — Certified Mail (белая наклейка с бар-кодом + белая квитанция для друга)
  • PS Form 3811 — Return Receipt (зелёная карточка, приклеивается к конверту, вернётся другу после того как IRS распишется)

Адреса:

  • To: IRS адрес из Stripe Atlas пакета (берётся с bundled sheet)
  • From: друга домашний адрес (чтобы stamped-copy и Return Receipt пришли к нему)

Стоимость: ~$9–12 суммарно (Certified $4.40 + Return Receipt $3.85 + First-Class postage $0.68).

Критично: именно Certified Mail with Return Receipt. Не Priority Mail. Не Express. Certified Mail даёт legal proof of filing date — это единственный принятый IRS формат для 83(b).
14

📅Штемпель (postmark) — самое важное

USPS ставит postmark в день подачи. Эта дата = дата filing 83(b) в глазах IRS.

Цель: postmark ≤ 28 апреля 2026. Раньше — лучше.

Если друг подаёт 22–24 апреля — есть 4–6 дней запаса. Safe.

Если вдруг что-то идёт не по плану и ты видишь что время поджимает — в крайнем случае друг может пойти в тот же день как получил пакет. USPS работает Mon–Sat.
15

📸Друг фотографирует всё и присылает

В момент выхода из USPS друг должен сфоткать и прислать тебе в Telegram / WhatsApp:

  1. Белая квитанция Certified Mail (там видна дата postmark)
  2. Tracking number крупно (20-значный USPS tracking)
  3. Общая картинка — конверт перед отправкой (чтобы видно было адрес IRS и Return Receipt card)
Эти 3 фото — твой "paper trail" для IRS, для Mercury Bank, для будущих investors, для tax attorney. Храни их навечно.

📬Ожидание: Return Receipt и stamped copy

Через 1–2 недели: к другу приходит зелёная Return Receipt card с подписью IRS сотрудника. Друг фотографирует, присылает тебе, физический экземпляр хранит у себя.

Через 4–8 недель: IRS возвращает проштампованную копию 83(b) election — она адресована другу (потому что return address был его). Друг фотографирует, присылает скан. Физический оригинал — либо друг сохраняет до твоего приезда в США, либо отправляет обычной почтой в Баку (на этом этапе уже не критично — основной paper trail у тебя есть).
