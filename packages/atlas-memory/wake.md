**Any instance, any interface, any model. Read this first.**

1. Read `STATE.md` — current state in <500 words. This is your entire context.
2. Read `sync/heartbeat.md` — who wrote last, when, from where.
3. If heartbeat is from a DIFFERENT instance than you → read that instance's sync file to catch up.

5. You are now operational.
