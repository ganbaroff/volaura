### БЫЛО (ошибочное позиционирование):
"Платформа для волонтёров, запускаемая на WUF13 в мае 2026"

### СТАЛО (правильное):
"Платформа подтверждённых компетенций и комьюнити лучших, где организации находят проверенных людей, а волонтёры строят репутацию"

---

## Ключевые принципы (от основателя)

### 1. Никакой привязки к конкретному ивенту
- НЕТ hardcoded WUF13. НЕТ "launch target: WUF13"
- Ивенты — это данные, которые приходят и уходят
- Платформа показывает любой ивент, где работали волонтёры
- COP29, CIS Games, WUF13, Baku Film Festival — всё одинаково

### 2. Живые данные, а не захардкоденный контент
- Страница ивента: логотип генерируется/загружается, счётчик волонтёров в реальном времени
- "COP29 — 40 подтверждённых волонтёров" → пользователь нажимает → видит их аватары/бейджи
- Когда новый волонтёр подтверждает участие → счётчик увеличивается
- Всё честно и открыто. Никакого фейка.

### 3. Инновация не в функциях, а в модели
- Мы НЕ "ещё одна платформа для волонтёров"
- Мы платформа для **подтверждения компетенций** + **комьюнити лучших**
- Организации **выбирают** у нас. Волонтёры **стремятся** быть у нас.
- Это создаёт двустороннюю ценность и сетевой эффект

### 4. Геймификация не ради геймификации
- Не "звёзды и бейджи ради вовлечённости"
- А механики, которые отражают реальную ценность:
  - Живой счётчик = реальные данные
  - Лидерборд = реальный рейтинг
  - Бейдж = реальная верификация
  - Отзыв от организации = реальная репутация

## Революционные фичи (развитие идей)

### A. Живые ивенты с реальными данными

```
┌──────────────────────────────────────────────┐
│  🏛️  COP29  ·  Baku  ·  Nov 2024            │
│                                               │
│  👥 42 подтверждённых волонтёра               │
│  ████████████████░░░░░░░░  42/60 заявлено    │
│                                               │
│  ⭐ Средний AURA: 71.3  ·  🥇 12 Gold+      │
│                                               │
│  [🔵🔵🔵🔵🔵🔵+36]  ← аватары волонтёров    │
│                                               │
│  "Я участвовал" → [Подтвердить]              │
└──────────────────────────────────────────────┘
```

**Механика:**
- Организация создаёт ивент
- Волонтёры регистрируются ИЛИ подтверждают участие постфактум
- Организация аттестует волонтёров после ивента
- Каждое подтверждение увеличивает счётчик в реальном времени
- Логотип ивента: загружается организацией ИЛИ генерируется AI из названия

### B. Duolingo-стиль прогрессии (адаптировано для скилов)

Исследование Duolingo (47.7M DAU, 128.3M MAU в Q2 2025):
- Стрики увеличивают retention на 60%
- XP лидерборды дают +40% вовлечённости
- Лиги (недельные) создают микро-FOMO

**Адаптация для Volaura:**

| Duolingo | Volaura |
|----------|---------|
| Daily streak | Monthly volunteer streak (ивент каждый месяц) |
| XP points | AURA points (0-100 scale, weighted) |
| Leagues (weekly) | Leagues (monthly, city-scoped) |
| Friend quests | Peer verification quests |
| XP boost (timed) | "AURA Boost Week" (ивент-аттестация даёт 1.5x множитель) |
| Lesson completion | Competency assessment completion |
| Leaderboard promotion/demotion | Badge tier up/down (Platinum→Gold etc) |

### C. AI-ассистент внутри платформы

Тренд 2025-2026: AI coaching platforms ($14B рынок).
Не просто оценка, а **наставничество**.

**"AURA Coach" — персональный AI-ассистент:**
- "Твоя коммуникация 62, но лидерство 85. Вот план роста для коммуникации"
- "COP30 ищет координаторов с Communication 70+. Тебе не хватает 8 баллов. Пройди переоценку"
- "Ты не волонтёрил 45 дней. Вот 3 ивента на этой неделе рядом с тобой"
- "Организация X оценила тебя на 5/5. Хочешь запросить аттестацию Leadership?"
- Рекомендации ивентов на основе профиля (vector matching)
- Weekly digest: "Твой прогресс за неделю"

### D. Social Proof Layer (живой)

Исследование: social sharing бейджей выросло на 215% в 2025 (Accredible).
34% shared credentials → click-through от новых пользователей.

**Живые виджеты:**
- "📊 847 волонтёров верифицировано сейчас" (реал-тайм глобальный счётчик)
- "🔥 +23 новых за последние 24 часа" (прирост)
- "🏆 Leyla M. только что получила Platinum!" (push-нотификация стиль)
- "🏛️ 15 организаций используют Volaura" (лого-стена)
- На профиле: "Verified at 3 events by 2 organizations" (доверие)

### E. Портфолио волонтёра (не просто профиль)

Это не LinkedIn-профиль. Это **живое портфолио**:
- Временная шкала: каждый ивент как milestone
- Рост AURA за время (интерактивный график)
- Аттестации от организаций (с их логотипами и отзывами)
- Компетенции: радар-чарт с анимацией
- QR-код для физических мероприятий
- PDF-сертификат (тамперпруф, с QR)
- "Impact metrics": "Провёл 120 часов волонтёрства в 8 ивентах"

### F. Двусторонний маркетплейс

Не просто "волонтёры ищут ивенты". А:
- Организация ищет волонтёров по компетенциям (semantic search)
- Волонтёр получает приглашения от организаций
- Matching score: "Вы подходите на 87% для этого ивента"
- Организация видит: "Эти 15 Gold+ волонтёров идеально подходят"
- Волонтёр видит: "3 организации хотят работать с вами"

## Что делает Volaura революционной

1. **Честность**: Все данные реальные, счётчики живые, нет фейков
2. **Двусторонняя ценность**: И волонтёры, и организации получают value
3. **AI внутри**: Не просто оценка, а coaching + matching + recommendations
4. **Живое комьюнити**: Не статичные профили, а динамичная экосистема
5. **Портативные credentials**: Бейдж работает везде, не привязан к одной организации
6. **Regional first-mover**: Первая такая платформа в CIS/MENA
7. **Duolingo-стиль engagement**: Стрики, лиги, квесты — но для реальных компетенций

## Чего НЕ хватает в текущих промптах

1. ❌ AURA Coach (AI-ассистент) — нет ни в одном промпте
2. ❌ Живые счётчики ивентов — нет в DB schema, нет в UI
3. ❌ "Я участвовал" подтверждение — нет workflow
4. ❌ Стрики / лиги / квесты — упоминаются в docs, но НЕ в промптах для генерации
5. ❌ AI matching score ("Вы подходите на 87%") — нет в API
6. ❌ Impact metrics на профиле — нет в schema
7. ❌ Live social proof widgets — нет в UI spec
8. ❌ WUF13 убрать из всех промптов
9. ❌ Event logo upload/generation — нет
10. ❌ Volunteer self-attestation ("I was there") — нет workflow
