## 2.4 K-Factor Analysis (Realistic)

```
K-factor = i x c

i = number of invites/shares per user
c = conversion rate of invites to signups
```

```
Scenario 1: Badge sharing on LinkedIn
  i = 2 (badge post reaches ~2 people who actually click through)
  c = 0.05 (5% of clickers sign up -- low, no social proof yet)
  K = 2 x 0.05 = 0.10

Scenario 2: Founder LinkedIn content
  i = 5 (good post reaches 500 people, 1% click = 5)
  c = 0.03 (3% of visitors sign up)
  K = 5 x 0.03 = 0.15
```
