Glossary — ликбез параллельно

Технические термины простыми словами — чтобы ты видел не только статус, но и понимал, о чём я говорю.
Railway
Хостинг бэкенда (Python/FastAPI). Примерно как Heroku. У нас там volauraapi-production.up.railway.app.
Vertex AI
Google Cloud API для Gemini. У тебя $300 стартовых кредитов, сейчас их используем. Платная ветка — быстрее и лимиты выше.
AI Studio Gemini
Бесплатная ветка того же Gemini с жёсткими rate-лимитами. Наш fallback, когда Vertex упал.
Fallback chain
Цепочка резервирования провайдеров. Первый упал → автоматом идёт на второй. У нас: Vertex → AI Studio → Groq → OpenAI. Юзер не замечает падения.
CI (Continuous Integration)
GitHub Actions. При git push GitHub автоматически гоняет тесты и деплоит. Тесты упали — деплой не проходит.
GitHub Secrets
Сейф внутри GitHub для API-ключей. CI оттуда читает, в коде ключей нет.
Swarm / 13 perspectives
Автономная команда 13 виртуальных специалистов в packages/swarm/. Смотрят на код с разных углов (security, design, assessment, legal). Плюс ~118 skill-модулей — библиотека умений. НЕ 44 — цифра 44 была моей галлюцинацией из Session 93, сегодня зачистил.
Show-stopper
Баг, который полностью блокирует сценарий пользователя. Не косметика — "продукт не работает".
Propagation
Перенос ключа из локального .env на продакшен. Локально работает ≠ на сервере работает.
Sprint
Рабочий цикл. У нас номер сессии ≈ номер спринта. Сейчас 119-й.
GDPR consent_events
В ЕС юридически обязательно логировать каждое действие пользователя со согласиями. Таблица есть, 26 из 27 endpoints пишут мимо неё. Аудит = штраф.
Feature-audit
Полная инвентаризация: 50 фич реально работают, 24 частично, 13 "заявлены и не найдены", 11 в планах. Зеркало маркетинга vs реальность.
