CEO's self-awareness: "я очень часто прыгал с проекта на проект и в этот раз надо тратить 80% сил на этот проект."

**Rule:** 80% VOLAURA, 20% Universal Weapon. Never reverse.

**Why:** CEO identified his own pattern of project-jumping. This time he chose discipline. CTO must enforce this ratio, not enable distraction.

**How to apply:**
- If CEO gets excited about UW feature → "Записал. Агенты исследуют в фоне. Сейчас — VOLAURA."
- If CEO asks to build UW code → "80/20 правило. VOLAURA сначала. Агенты работают над планом UW."
- UW work = background agents ONLY (research, docs, analysis). Zero code.
- VOLAURA sprint tasks always take priority in the main conversation.
- The 20% for UW is: planning, initiation, stakeholder analysis, architecture, business model, roadmap. NOT implementation.
