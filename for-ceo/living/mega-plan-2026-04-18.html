4Жёсткие решения — что убираем, чтобы ехать

Каждое из этих решений проверено на критике выше. Если не сделать — инфраструктура будет расти быстрее, чем жители города.

Stop · 3 продукта замораживаются

Life Simulator, BrandedBy, ZEUS → archive/ до сигнала от M3. Не удаляем, не мигрируем, не развиваем. Все силы Atlas на 1–2 продукта.

Stop · новые .md файлы без living-doc

Правило update-don't-create становится хуком: pre-commit проверяет, что новый .md не создан без announce «phase change». Всё остальное — Edit в существующий.

Stop · инфра без external validation

Любой план > 2 дней работы проходит Doctor Strange v2 Gate 1 (внешняя модель) до первой строки кода. Без этого — не начинаем.

Start · weekly user-interview × 5

С M3 — 5 живых пользователей в неделю, 30 мин каждый. Без этого любая метрика на /admin — это метрика для одного юзера (тебя), и она врёт.

Start · ключи AWS + Google Cloud

Bedrock (Claude), Vertex AI (Gemini Pro 2.5), SES для email, SageMaker для LoRA. Credit-capital сейчас лежит без дела. Проактивно беру, per новый Proactive-credits rule.

Start · metrics as pain-signals

Каждая новая метрика рождается с вопросом «как ноцицептор для этой боли должен молчать в норме». Иначе не попадает в /admin.
