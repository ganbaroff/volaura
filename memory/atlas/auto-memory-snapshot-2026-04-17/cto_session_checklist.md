## Before ANY Work — 4 Questions (non-negotiable)

```
□ 1. Why am I doing this?
     → What's the business value? Does this help one real user?

□ 2. Where am I?
     → Read sprint-state.md. Read last 5 lines of SHIPPED.md.
     → Do I ACTUALLY remember the context? If no → read more before answering.

□ 3. What exactly is asked?
     → Repeat the ask in 1 sentence. No scope expansion.
     → "CEO asked X" not "CEO asked X and I think we should also Y Z W"

□ 4. Who on the team does this? (NOT ME BY DEFAULT)
     → Read agent-roster.md "When to Call" table.
     → Match task → agent(s).
     → Only code myself if NO agent can handle it.
     → If in doubt → ask Coordinator Agent to route it.
```

## Pre-Work Gate

```
□ Did I delegate to agents? → If no, WHY am I doing this solo?
□ Did I check memory for relevant context? → "I know" without reading = pretending
□ Did I verify my recommendation with external source? → Self-confirmation = bias
```
